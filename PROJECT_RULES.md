# LuxeForge Studio Rules

1. Iedere commit moet werkend zijn.
2. Geen code zonder type hints.
3. Iedere functie heeft een docstring.
4. Geen Blender operators in de Geometry Engine.
5. Alles parametrisch.
6. STL's worden gegenereerd, niet aangepast.
7. Code wordt modulair gehouden.
8. Iedere feature krijgt een GitHub Issue.
