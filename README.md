# LuxeForgeStudio

LuxeForge Studio is a professional Blender add-on for creating luxury-inspired, 3D-printable accessories using fully parametric geometry.

## Features (Roadmap)

- Parametric handbag generator
- Quilting engine
- Chain generator
- Hardware generator
- Print analyzer
- STL / 3MF export
- Production dashboard

## Current Version

v0.1.0-alpha.1
